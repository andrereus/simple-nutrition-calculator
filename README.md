## Setup

```
npm install
```

Activate EditorConfig, ESLint, Stylelint and Prettier in your Editor / IDE.

## Usage

### For development

```
npm run dev
```

In a new terminal run:

```
npm run dev:pwa
```

### For deployment

```
npm run build

npm run deploy
```

Optional:

```
now alias https://build-name.now.sh domain-name.com
```
